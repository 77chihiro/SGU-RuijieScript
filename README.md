# SGU-RuijieScript韶关学院电信第三方拨号器（锐捷）
感谢工信部对大学生们的监督，国内的校园网有了一人一号的限制。在某些企业的变本加厉之下，甚至已经演变为了一设备一号。将校园网与设备进行绑定吃相属实。。。（自行脑补）。但是我的网络设备有点多所以我需要一个满足我需求的网络环境。
